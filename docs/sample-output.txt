SQLITE VERIFICATION COMMANDS
======================================

After running a scan:

  # Check all findings for the latest scan cycle
  sqlite3 backend/data/marketmind.db \
    "SELECT capability, verdict, confidence FROM findings ORDER BY created_at DESC LIMIT 10"

  # Check signal ranker output (is_new field)
  sqlite3 backend/data/marketmind.db \
    "SELECT capability, score, is_new FROM findings WHERE capability LIKE 'scout-%' ORDER BY score DESC"

  # Check latest brief
  sqlite3 backend/data/marketmind.db \
    "SELECT executive_summary, confidence FROM briefs ORDER BY created_at DESC LIMIT 1"

Expected: 7 findings per scan cycle (4 scouts + ranker + analyst + strategist)
Ranker finding should have score > 0 and is_new field populated
